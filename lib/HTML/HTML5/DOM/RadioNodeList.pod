=head1 NAME

HTML::HTML5::DOM::RadioNodeList - implementation of the RadioNodeList interface of the HTML DOM

=head1 DESCRIPTION

HTML::HTML5::DOM::RadioNodeList is an implementation of the RadioNodeList interface of the HTML DOM. See L<HTML::HTML5::DOM> for a list of the conventions that have been used when translating the DOM to Perl.
