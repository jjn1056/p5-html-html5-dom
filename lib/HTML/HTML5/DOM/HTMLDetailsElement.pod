=head1 NAME

HTML::HTML5::DOM::HTMLDetailsElement - implementation of the HTMLDetailsElement interface of the HTML DOM

=head1 DESCRIPTION

HTML::HTML5::DOM::HTMLDetailsElement is an implementation of the HTMLDetailsElement interface of the HTML DOM. See L<HTML::HTML5::DOM> for a list of the conventions that have been used when translating the DOM to Perl.

=head2 HTML Elements

This class applies to the following HTML elements.

=over

=item * C<< {http://www.w3.org/1999/xhtml}details >>

=back

=head2 Inheritance

HTML::HTML5::DOM::HTMLDetailsElement inherits methods from the following Perl classes.

=over

=item * L<HTML::HTML5::DOM::HTMLElement>

=item * L<XML::LibXML::Augment::Element>

=item * L<XML::LibXML::Augment::Node>

=item * L<XML::LibXML::Element>

=item * L<XML::LibXML::Node>

=item * L<XML::LibXML::QuerySelector>

=item * L<HTML::HTML5::DOMutil::FancyISA>

=back

=head2 Additional Methods

As well as its inherited methods, this class provides the following methods.

=over

=item * C<< open >>

Called with no arguments, is a shortcut for C<< $elem->hasAttribute("open") >>. If called with a true argument, will C<setAttribute>; if called with a false argument will C<removeAttribute>.

=back
