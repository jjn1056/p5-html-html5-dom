=head1 NAME

HTML::HTML5::DOM::HTMLMediaElement - implementation of the HTMLMediaElement interface of the HTML DOM

=head1 DESCRIPTION

HTML::HTML5::DOM::HTMLMediaElement is an implementation of the HTMLMediaElement interface of the HTML DOM. See L<HTML::HTML5::DOM> for a list of the conventions that have been used when translating the DOM to Perl.

=head2 HTML Elements

This class applies to the following HTML elements.

=over

=back

=head2 Inheritance

HTML::HTML5::DOM::HTMLMediaElement inherits methods from the following Perl classes.

=over

=item * L<HTML::HTML5::DOM::HTMLElement>

=item * L<XML::LibXML::Augment::Element>

=item * L<XML::LibXML::Augment::Node>

=item * L<XML::LibXML::Element>

=item * L<XML::LibXML::Node>

=item * L<XML::LibXML::QuerySelector>

=item * L<HTML::HTML5::DOMutil::FancyISA>

=back

=head2 Additional Methods

As well as its inherited methods, this class provides the following methods.

=over

=item * C<< controls >>

Called with no arguments, is a shortcut for C<< $elem->getAttribute("controls") >>. Called with a defined argument, acts as C<setAttribute>. Called with undef as an argument, acts as C<removeAttribute>.

=item * C<< crossOrigin >>

Called with no arguments, is a shortcut for C<< $elem->getAttribute("crossorigin") >>. Called with a defined argument, acts as C<setAttribute>. Called with undef as an argument, acts as C<removeAttribute>.

=item * C<< preload >>

Called with no arguments, is a shortcut for C<< $elem->getAttribute("preload") >>. Called with a defined argument, acts as C<setAttribute>. Called with undef as an argument, acts as C<removeAttribute>.

=item * C<< src >>

Shortcut for C<< HTTP::Request->new(GET => $elem->href) >>

=back
